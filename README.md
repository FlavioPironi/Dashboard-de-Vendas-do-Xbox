# Dashboard-de-Vendas-do-Xbox

Um dashboard de vendas, com foco na organização e visualização de dados.

Este projeto foi desenvolvido como exercício no curso Excel com IA na DIO através de uma parceria com a Santander Open Academy Nele temos uma planilha interativa em Excel.

Descrição do Desafio: O objetivo é criar um dashboard de vendas, com foco na organização e visualização de dados. Transformando dados brutos em informações visuais claras e úteis, permitindo uma análise eficaz do desempenho de vendas e a tomada de decisões baseadas em dados.
